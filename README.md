# FrameFlow Pro

一款功能强大的视频抽帧工具，支持批量提取视频帧和单帧捕捉。

## 功能特性

- 🎬 **视频预览** - 支持导入多种视频格式进行实时预览
- ▶️ **播放控制** - 播放/暂停视频，通过时间轴快速定位
- 📸 **单帧捕捉** - 一键捕捉当前帧并保存为图片
- 🔄 **批量抽帧** - 自定义起始帧、结束帧和抽帧间隔，批量提取视频帧
- 📊 **实时进度** - 显示处理进度和预计提取帧数
- 🎨 **现代界面** - 采用 Shadcn UI 风格设计，简洁美观

## 安装

### 环境要求

- Python 3.7+
- Windows / macOS / Linux

### 安装依赖

```bash
pip install -r requirements.txt
```

或手动安装依赖：

```bash
pip install PySide6 opencv-python numpy
```

## 使用方法

### 启动程序

```bash
python app.py
```

### 操作步骤

1. **导入视频** - 点击"导入视频"按钮选择视频文件
2. **预览视频** - 使用播放/暂停按钮和时间轴预览视频内容
3. **标记范围** - 使用"标记起点"和"标记终点"按钮设置抽帧范围
4. **设置间隔** - 调整抽帧间隔（默认为 10 帧）
5. **选择保存路径** - 点击"更改目录"选择图片保存位置
6. **开始批量执行** - 点击"开始批量执行"按钮开始抽帧

### 单帧捕捉

在预览视频时，点击"捕捉当前帧"按钮即可保存当前帧为图片。

## 依赖项

- **PySide6** - Qt GUI 框架
- **opencv-python** - 视频处理库
- **numpy** - 数值计算库
